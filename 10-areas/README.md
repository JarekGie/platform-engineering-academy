# 10-areas

Trwałe obszary wiedzy/kompetencji przekrojowe wobec modułów (bez daty końca), w odróżnieniu
od [[20-projects|inicjatyw z terminem]]. W typie `academy` serce pracy jest w katalogach pipeline
(`01-roadmap` … `08-real-projects`); `10-areas/` jest lekki i opcjonalny.

Co tu trafia:
- przekrojowe obszary kompetencji spinające wiele modułów (np. „observability", „bezpieczeństwo"),
- trwałe notatki referencyjne niezwiązane z jednym modułem.
