# _system

Katalog systemowy vaulta. Generator (`bin/new-vault.sh`) KOPIUJE tu kontrakty LLM z `contracts/`
(m.in. `CLAUDE.md`, `AGENTS.md`, `CODEX.md`, kontrakty izolacji domen, NotebookLM, compliance, podcast).
Nie edytuj tych plików ręcznie w vaultcie — zmieniaj je w repozytorium frameworka.

Co tu trafia:
- kontrakty zachowania dla narzędzi LLM (kopiowane przez generator),
- reguły izolacji domen i granicy kontekstu,
- polityki ekspozycji i przeglądu compliance.
