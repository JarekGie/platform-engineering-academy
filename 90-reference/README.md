i eksperymentów. Materiał kursowy „do przerobienia" należy do vaulta `learning`; tu trzymaj
referencje wspierające budowę kompetencji i dowody.

Co tu trafia:

- artefakty źródłowe wskazywane przez `source_ref` z [[evidence-register]].
