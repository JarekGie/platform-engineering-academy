# NotebookLM — platform-engineering-academy

Katalog roboczy integracji z Google NotebookLM dla tego vaulta.

NotebookLM **nie jest** source of truth. Prawda zawsze zostaje w vaultcie
(notatki w 10-areas/, 20-projects/, 30-standards/, 40-runbooks/, 50-patterns/).
NotebookLM sluzy do eksploracji i generowania artefaktow (np. podcastow),
nigdy do przechowywania wiedzy zrodlowej.

## Co tu trafia
- `sources.md` — rejestr zrodel zgloszonych/zatwierdzonych do NotebookLM.
- `notebook-contract.md` — skrot kontraktu NotebookLM (pelny: [[NOTEBOOKLM_CONTRACT]]).
- `podcast-policy.md` — polityka eksportu podcastow (pelna: [[PODCAST_EXPORT_POLICY]]).
- `prompts/` — prompty uzyte w NotebookLM.
- `findings/` — ustalenia i wnioski z sesji.
- `artifacts/` — wygenerowane artefakty (audio, notatki, briefy).
- `conversations/` — zapisy rozmow z NotebookLM.
