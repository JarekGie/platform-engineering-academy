# 04-assessments

Krok 4 pipeline: **assessmenty**. Sprawdzenie wiedzy i umiejętności praktycznych. Każdy assessment
wspiera: questions · practical tasks · passing criteria · remediation plan. Nowe assessmenty skrobaj
z szablonu `templates/assessment.md`.

Pipeline: Roadmap → Theory → Lab → **Assessment** → Homelab → Evidence → Real Project.
