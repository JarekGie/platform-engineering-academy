# Przykład — source-pack dla podcastu „Theory → Evidence"

Skrót zebrania źródeł dla odcinka serii How-To (Podcast 3). Kluczowe: **tylko źródła `approved`,
tylko jedna domena, twardy gate bezpieczeństwa przed eksportem.**

```
User:  następny podcast
Skill: Seria How-To, kolejny to „Theory → Evidence". Zbieram źródła (approved, domena learning)...
Skill: Znalazłem 2 kandydatów approved: [[roadmap]], [[evidence-register]].
       Odrzucone: notatka real-project (classification: confidential) — poza eksportem.
```

Wynikowy `source-pack.md` wskazuje `[[roadmap]]` i `[[evidence-register]]` (wskaźniki, nie treść),
wszystkie w domenie `learning`, `podcast_export: allowed`. Brief i glossary powstają z tych źródeł.
Odcinek trafia do `academy-how-to/`, feedback po odsłuchu — do `feedback/`.
