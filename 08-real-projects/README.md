# 08-real-projects

Ostatni krok pipeline: **rzeczywiste projekty** — końcowy dowód kompetencji. Projekty kończące moduły,
integrujące wiele kompetencji, z lessons learned, final evidence i artefaktami portfolio-grade.

Świadomy rozdział: projekty demonstracyjne i środowisko treningowe to [[03-labs]] / [[07-homelab]];
tutaj trafiają **rzeczywiste wdrożenia i case studies**. Nie mieszaj jednych z drugimi.

Real project produkuje final evidence (`source_system: academy`) konsumowane przez Career OS.

Pipeline: Roadmap → Theory → Lab → Assessment → Homelab → Evidence → **Real Project**.
