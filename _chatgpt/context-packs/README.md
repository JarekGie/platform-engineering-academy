# context-packs

Konkretne paczki kontekstu generowane skillem context-pack do wklejenia w zewnętrzne narzędzie AI.
Każda paczka jest samodzielna i zgodna z `llm_exposure` źródłowych notatek.

Co tu trafia:
- pliki paczek (np. `kubernetes-module-pack` pod naukę modułu),
- metadane: zakres, data, klasyfikacja,
- zarchiwizowane, nieaktualne paczki.
