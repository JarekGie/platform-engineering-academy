# _chatgpt

Katalog na materiały przygotowane do użycia w zewnętrznym ChatGPT (poza Claude Code).
Trzyma gotowe paczki kontekstu (context-packs) świadomie wyeksportowane z vaulta.
Academy jest `internal` i publikowalna — mimo to przed eksportem sprawdź `llm_exposure`
i `classification` notatek źródłowych.

Co tu trafia:
- paczki kontekstu w [[context-packs]] (np. pod naukę konkretnego modułu),
- notatki o zakresie i dacie eksportu.
