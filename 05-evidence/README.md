# 05-evidence

Krok 5 pipeline: **dowody kompetencji**. Academy jest **PRODUCENTEM** evidence; Career OS jest
konsumentem. Główna notatka: [[evidence-register]].

Zasada evidence-first: każda deklarowana kompetencja ma dowód (lab / assessment / homelab / real project).
Federacja z Career OS przez `evidence_id` + `source_system: academy` + `source_ref` + `verification_status`
— bez duplikacji treści.

Pipeline: Roadmap → Theory → Lab → Assessment → Homelab → **Evidence** → Real Project.
