# 07-homelab

Krok „środowisko treningowe" pipeline: **homelab** jako obywatel pierwszej klasy. Tu buduje się
i utrzymuje środowisko, w którym wykonywane są laby. Wspiera: architekturę, ADR, diagramy, adresację,
provisioning, platformę i eksperymenty.

Główne notatki: [[homelab-architecture]], [[homelab-adr]]. Homelab jest **treningiem** — rzeczywiste
wdrożenia i case studies należą do [[08-real-projects]].

Pipeline: Roadmap → Theory → Lab → Assessment → **Homelab** → Evidence → Real Project.
